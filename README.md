# Run Locally
Clone the project
```bash
  git clone https://github.com/MohammadAliAlizada/youtube-mern-stack.git
```
Go to the project directory
```bash
  cd youtube-mern-clone
```
Install dependencies
```bash
  npm install
```

Start the server
```bash
  npm start
```
Start the Client
