these technologies are used!

# Run Locally
Clone the project
```bash
  git clone https://github.com/MohammadAliAlizada/portfolio.git
```
Go to the project directory
```bash
  cd portfoli
```
Install dependencies
```bash
  npm install
```

Start the server
```bash
  npm start
```
Start the Client
